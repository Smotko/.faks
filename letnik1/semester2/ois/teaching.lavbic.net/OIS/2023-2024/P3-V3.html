Poglavje 6 P3 → V3 JavaScript na strani odjemalca

P3 → V3

V pripravah na vaje V3 bo predstavljen kratek uvod v:

  • JavaScript (načini vpeljave JavaScript kode, spremenljivke, logične operacije, vejitve, zanke, funkcije, objekti, seznami, DOM, dogodki in povratni klici, AJAX, JSON, kontekst, čakanje, da se stran naloži).

Celotna izvorna koda primera v nadaljevanju je na voljo v Github repozitoriju P3-V3. Najprej na Github izdelajte kopijo projekta (angl. Fork), nato pa si lokalno okolje nastavite z naslednjima ukazoma:

git clone git@github.com:{VAŠE-GITHUB-UPORABNIŠKO-IME}/P3-V3.git
cd P3-V3

ali

git clone https://github.com/{VAŠE-GITHUB-UPORABNIŠKO-IME}/P3-V3.git
cd P3-V3

6.1.1.9 Čakanje, da se stran naloži

Pogosto je treba imeti celotno stran naloženo (DOM drevo zgrajeno v celoti), da lahko manipuliramo z vsebino in takrat moramo počakati, da se stran naloži do konca.

koda.js

window.addEventListener("load", function() {
  console.log("Stran se je v celoti naložila.");
});

6.1.1.11 Dogodki in povratni klici

Večino programske kode v brskalniku prožijo dogodki in programiranje v JavaScriptu je pogosto čakanje na določen dogodek in reagiranje nanj. Dogodki se zgodijo npr. ob nalaganju strani, ko je uporabnik v interakciji s stranjo (npr. kliki, spremembe itd.).

Dodajmo najprej nov gumb za preverjanje podatkov:

index.html

<p>
  <input type="submit" value="Pošlji podatke">
  <input type="button" value="Preveri podatke">
</p>

in nato sprogramirajmo njegov odziv:

koda.js

var obdelajKlik = function() {
  console.log("Potrebno je preveriti vnesene podatke!");
};
var gumb = document.querySelector("input[type='button']");
gumb.addEventListener("click", obdelajKlik);

JavaScript konzola (ob kliku na gumb)

Potrebno je preveriti vnesene podatke!

6.1.1.12 AJAX

Če želimo v okviru strani pridobiti novo vsebino, uporabimo XHR (XML HTTP Request). Spletne strani, ki uporabljajo ta pristop, se pogosto imenujejo AJAX (Asynchronous JavaScript and XML) aplikacije.

Dejansko vse spletne strani, ki posodabljajo vsebino obstoječe strani (npr. Facebook, Gmail, Google Maps itd.), uporabljajo to tehniko. Uvedel jo je Microsoft pri razvoju Outlook Web Access aplikacije.

koda.js

var req = new XMLHttpRequest();
req.onload = function () {
  console.log("Podatki oddaljene storitve pridobljeni!");
  console.log(this.responseText);
};
req.open(
  "get",
  "https://teaching.lavbic.net/api/kraji/iskanje/postnaStevilka/3000",
  true
);
req.send();

JavaScript konzola

Podatki oddaljene storitve pridobljeni!
[{"postnaStevilka":3000,"kraj":"Celje - dostava","lat":45.5916,"lng":14.1546}]

Najprej kreiramo objekt XMLHttpRequest, nato določimo povratni klic z req.onload, ki se izvede, ko je klic zaključen. Z ukazom req.open povemo odjemalcu, da bi želeli odpreti povezavo do željenega spletnega naslova in zadnji parameter pove, ali je klic asinhron (če je, se lahko ostala koda izvaja naprej).

6.1.1.13 JSON

JSON zapis je množica pravil za zapis in prenos podatkov v obliki, berljivi uporabnikom in računalniku. Je zelo podoben JavaScript objektu, od koder tudi izhaja.

var anakondaNiz = JSON.stringify({
  kaca: "Anakonda",
  habitat: "Južna Amerika",
  dolzina: "5-7 m"
});
console.log(anakondaNiz);
var anakondaObjekt = JSON.parse(anakondaNiz);
console.log(anakondaObjekt.dolzina);

JavaScript konzola

{"kaca":"Anakonda","habitat":"Južna Amerika","dolzina":"5-7 m"}
5-7 m

Če podatke prejemamo v obliki niza, potem ponavadi niz pretvorimo v JSON obliko z ukazom JSON.parse(), če pa jih pošiljamo, potem izvedemo obratno operacijo z ukazom JSON.stringify().

6.1.1.14 Kontekst

Kontekst oz. vidnost spremenljivk je v mnogo programskih jezikih opredeljena z blokom, kjer spremenljivke opredelimo.

6.1.1.14.1 Kontekst funkcije

Pri JavaScriptu so stvari nekoliko drugačne, saj uporablja kontekst funkcije. To pomeni, da spremenljivke niso vidne zunaj funkcije, kjer smo jih opredelili. Če niso opredeljene znotraj funkcije, potem so na voljo globalno.

koda.js

var narediNekaj = function() {
  var a = 10;
};
narediNekaj();
console.log(a);     // a ni opredeljen, pride do napake

  var c = 5;
