Celotna izvorna koda primera v nadaljevanju je na voljo v Github repozitoriju P2-V3. Najprej na Github izdelajte kopijo projekta (angl. Fork), nato pa si lokalno okolje nastavite z naslednjima ukazoma:

git clone git@github.com:{VAŠE-GITHUB-UPORABNIŠKO-IME}/P2-V3.git
cd P2-V3

ali

git clone https://github.com/{VAŠE-GITHUB-UPORABNIŠKO-IME}/P2-V3.git

4.1 HTML

Za uvod v jezik HTML si lahko pogledate dodatno praktično gradivo HTML tutorial.

Spremembe v repozitoriju od tega koraka naprej so na voljo na uveljavitvi s sporočilom HTML gradniki.

4.1.1 Gradniki

4.1.1.1 Prosto besedilo

Kreiramo datoteko index.html in vstavimo naslednje besedilo:

index.html

Pozdravljeni v prodajalni bučk in divjih kač.

4.1.1.2 Oznake, atributi in elementi

Dokument spremenimo, da dobi naslednjo obliko:

index.html

<!DOCTYPE html>
<html>
  <body>
    Pozdravljeni v prodajalni bučk in divjih kač.
  </body>
</html>

<!DOCTYPE html> je deklaracija tipa dokumenta in pove brskalniku tip HTML dokumenta (v tem primeru HTML5).

Oznaki <html> in </html> predstavljata HTML dokument, medtem ko se bo vsebina (oz. oznaka) med <body> in </body> prikazala v brskalniku.

4.1.1.3 Naslov strani

Dodamo naslov strani:

index.html

<!DOCTYPE html>
<html>
  <head>
    <title>Prodajalna dobrot</title>
  </head>
  <body>
    Pozdravljeni v prodajalni bučk in divjih kač.
  </body>
</html>

Oznaka <head> se nahaja pred oznako <body> in vsebuje metapodatke o spletni strani.

4.1.1.4 Odstavki

Če na spletno stran dodamo novo vrstico, brskalnik tega ne bo pravilno interpretiral, ker po navadi podvojenih presledkov in prehodov v novo vrstico znotraj elementa ne upošteva.

index.html

<body>
  Pozdravljeni v prodajalni bučk in divjih kač.
  Na voljo so dodatni popusti.
</body>

Zato moramo uporabiti odstavke <p>...</p>. Na voljo pa imamo tudi odebeljeno <strong>...</strong> in ležeče besedilo <em>...</em> ter prelome vrstic <br/>.

index.html

<body>
  <p><em>Pozdravljeni</em> v prodajalni<br/> bučk in<br/> divjih kač.</p>
  <p>Na voljo so <strong>dodatni popusti</strong>.</p>
</body>

4.1.1.5 Naslovi

Na voljo imamo več ravni naslovov: <h1>, <h2>, <h3>, <h4>, <h5> in <h6>.

index.html

<body>
  <h1>Spletna trgovina</h1>
  <h2>Uvodni nagovor</h2>
  <p><em>Pozdravljeni</em> v prodajalni<br/> bučk in<br/> divjih kač.</p>
  <p>Na voljo so <strong>dodatni popusti</strong>.</p>
  <h2>Kontaktni podatki</h2>
  <p>+386 (0)90 90 90 90</p>
</body>

4.1.1.6 Seznami

Na voljo imamo neurejene (angl. unordered lists ul) in urejene sezname (angl. ordered lists ol):

index.html

<body>
  <h1>Spletna trgovina</h1>
  <h2>Uvodni nagovor</h2>
  <p><em>Pozdravljeni</em> v prodajalni<br/> bučk in<br/> divjih kač.</p>
  <p>Prodajamo predvsem</p>
  <ul>
    <li>melone,</li>
    <li>hokaido buče,</li>
    <li>anakonde in</li>
    <li>pitone.</li>
  </ul>
  <p>Specializirani pa smo na</p>
  <ol>

4.1.1.7 Povezave

Povezave v HTML jeziku opredelimo z oznako a in vsebuje atribut href, s katerim povemo ciljno mesto, kamor povezava kaže.

index.html

<ul>
  <li>melone,</li>
  <li>hokaido buče,</li>
  <li><a href="https://en.wikipedia.org/wiki/Anaconda">anakonde</a> in</li>
  <li>pitone.</li>
</ul>

4.1.1.8 Slike

Slike dodajamo s pomočjo gradnika img, kjer v okviru atributa src podamo povezamo do slike, ki se mora prikazati.

index.html

<p>Specializirani pa smo na</p>
<ol>
  <li>anakonde in</li>
  <li>melone.</li>
</ol>
<img src="anakonda.jpg" alt="Lovljenje anakonde">
<p>Na voljo so <strong>dodatni popusti</strong>.</p>

4.1.1.9 Tabele

Uporabimo oznako <table> za opredelitev tabele in oznako <tr> (angl. table row) za opredelitev vrstice ter oznako <td> (angl. table data) za opredelitev celice v tabeli.

index.html

<p>Na voljo so <strong>dodatni popusti</strong>.</p>
<table>
  <tr>
    <td>več kot 3 anakonde</td>
    <td>10% popust</td>
  </tr>
  <tr>
    <td>več kot 75 melon</td>
    <td>20% popust</td>
  </tr>
</table>

4.1.1.10 Obrazci

Za vnos podatkov uporabnikov lahko uporabimo oznako <form> in nato ostale elemente vnosa <input>, <textarea>, <select> itd. Pomembno je, da pri vsakem vnosnem polju vključimo tudi atribut name, ki mora biti na ravni obrazca enoličen.

4.1.1.11 SPAN in DIV

V jeziku HTML ima večina oznak svoj pomen (npr. <p> je odstavek, <h1> je naslov itd.). Na voljo imamo tudi oznaki <span> in <div>, ki nimata določenega pomena. Uporabljata se predvsem za grupiranje delov HTML kode skupaj, ki jih potem po navadi označimo z atributoma class ali id.

4.1.1.12 Meta oznake

Meta oznake <meta> ne zadevajo neposredno vsebine spletnih strani, ampak podrobneje opisujejo stran za lažjo interpretacijo s strani brskalnika oz. spletnih iskalnikov.

index.html

  <title>Prodajalna dobrot</title>
  <meta charset="utf-8">
  <meta name="description" content="Najboljša trgovina eksotičnih stvari v Sloveniji">

4.2 CSS

Za uvod v tehnologijo CSS si lahko pogledate dodatno praktično gradivo CSS tutorial.

4.2.1 Gradniki

Seznam vseh CSS lastnosti je na voljo v okviru uradne specifikacije.

4.2.1.1 Načini vpeljave slogov

4.2.1.1.1 Neposredno v elementu

index.html

<h1 style="color:DarkRed">Spletna trgovina</h1>
4.2.1.1.2 Stil znotraj dokumenta

index.html

<head>
  <title>Prodajalna dobrot</title>
  <meta charset="utf-8">
  <meta name="description" content="Najboljša trgovina eksotičnih stvari v Sloveniji">
  <style>
    h2 {
      color:DarkBlue;
    }
  </style>
</head>
4.2.1.1.3 Stil v zunanjem dokumentu

Kreiramo novo datoteko stil.css:

stil.css

em {
  color:DarkGreen;
}

in v datoteko index.html dodamo referenco:

index.html

<link rel="stylesheet" href="stil.css">
<style>
  h2 {
    color:DarkBlue;
  }
</style>

4.2.1.2 Selektorji, lastnosti in vrednosti

Na voljo imamo različne lastnosti. Eden izmed pomembnejših elementov je model škatle (obrobe, presledki, robovi ipd.). Uporabimo lahko tudi rezervirano besedo !important, s katero zahtevamo samodejno dedovanje lastnosti in zahtevamo, da se omenjeni stil vedno uporabi.

stil.css

img {
  padding:4px !important;
  border-radius:4px;
  border:1px solid #ddd;
}

index.html

stil.css

th, td {
